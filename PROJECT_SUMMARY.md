# 项目摘要：EdgeVision — 边云协同智能视觉检测系统

## 一、项目概述

**EdgeVision** 是一款面向工业自动化产线的智能视觉检测平台，融合传统图像处理与深度学习推理能力，采用可配置化 Pipeline 引擎和 MQTT 边云协同架构，旨在为中小型企业提供低成本、易部署、可扩展的视觉质检解决方案。

---

## 二、研究背景与动机

工业视觉检测是智能制造的核心环节。据统计，制造业中约 **30%** 的质量缺陷可通过视觉检测提前发现。然而，现有视觉检测工具面临三大挑战：

1. **检测流程固化**：更换产品需重新编写检测程序，开发周期长、成本高
2. **算法能力单一**：传统图像处理工具难以应对复杂场景，深度学习工具部署门槛高
3. **数据无法远程管控**：检测结果局限于本地，无法实现实时远程监控和数据分析

EdgeVision 针对以上痛点，提出了一套**"可配置化 Pipeline + 多模态算法融合 + 边云协同"**的综合解决方案。

## 三、系统设计

### 3.1 总体架构

系统采用**六层分层架构**，自上而下为：UI 层 → Widgets 层 → Controllers 层 → Core/Algorithm 层 → Config/Data 层 → Cloud Platform 层。各层职责清晰，依赖关系单向，确保系统的可维护性和可扩展性。

### 3.2 核心技术

| 技术方向 | 具体实现 | 创新点 |
|----------|----------|--------|
| **可配置化 Pipeline** | 9 步处理管线，步骤可独立配置 | 零代码搭建检测流程，实时预览效果 |
| **多模态检测** | 传统算法 + YOLO 深度学习 + 条码识别 | 三种检测能力融合，适应多种工业场景 |
| **GPU 加速推理** | ONNX Runtime + CUDA EP | YOLOv8n 推理速度 ~15ms/帧 |
| **边云协同** | MQTT 5.0 + Flask Web 看板 | 检测结果实时上报，云端远程监控 |
| **检测方案持久化** | ROI 归一化 + 参数序列化 + 模板库 | 跨设备复用，一键切换产品 |

### 3.3 Pipeline 引擎

Pipeline 引擎是系统的核心处理单元，由以下 9 个可配置步骤组成：

1. **通道转换**：Gray / RGB / HSV / 单通道
2. **图像增强**：亮度、对比度、Gamma 校正、锐化
3. **灰度过滤**：灰度阈值范围过滤
4. **颜色过滤**：RGB / HSV 色彩空间范围过滤
5. **形态学处理**：开闭运算、膨胀腐蚀、连通域分析、填充孔洞、面积筛选
6. **形状筛选**：面积、圆度、凸性、矩形度等多特征组合筛选
7. **直线检测**：HoughP / LSD / EDline 三种算法
8. **参考线匹配**：角度 + 距离容差匹配
9. **条码识别**：ZXing-CPP 多格式条码 / QR Code 识别

### 3.4 边云协同架构

- **边缘端**（C++ 桌面应用）：负责图像采集、Pipeline 处理、GPU 推理、结果判定
- **云端**（Python Flask）：负责数据接收、持久化存储、Web 可视化、远程指令下发
- **通信层**（MQTT 5.0）：支持检测结果上报、设备心跳保活、云端远程指令、自动重连

## 五、功能特性

1. **多图片管理**：单张/批量导入，多图片切换，独立 ROI 配置
2. **ROI 管理**：矩形绘制、多 ROI 支持、坐标归一化
3. **图像增强**：亮度/对比度/Gamma/锐化
4. **颜色过滤**：灰度/RGB/HSV 多模式
5. **算法处理队列**：10+ 种形态学操作自由组合
6. **形状提取与筛选**：多条件 AND/OR 逻辑
7. **直线检测**：三种算法 + 参考线匹配
8. **模板匹配**：多边形 ROI 提取，跨图片复用
9. **条码/QR Code 识别**：10+ 种格式支持
10. **深度学习目标检测**：YOLO 系列 ONNX 推理
11. **视频处理**：视频文件 + USB 摄像头实时检测
12. **批量检测**：自动遍历、实时统计、结果导出
13. **检测方案管理**：完整保存/加载，跨设备复用
14. **边云协同**：MQTT 实时上报 + 云端看板
15. **云平台看板**：设备监控、数据统计、远程指令

## 八、项目特色与创新

1. **可配置化 Pipeline 引擎**：将传统视觉检测的固定流程改为可自由组合的步骤链，用户无需编程即可搭建检测方案
2. **多模态算法融合**：在同一平台中融合传统图像处理、深度学习推理和条码识别三种能力
3. **边云协同架构**：基于 MQTT 5.0 协议实现边缘端与云端的实时通信，支持远程监控和控制
4. **检测方案持久化**：ROI 坐标归一化 + 完整参数序列化，实现跨设备、跨分辨率的方案复用
5. **工业级线程安全设计**：互斥锁 + 原子标志 + 值拷贝交换，确保 Pipeline 后台执行与 UI 响应互不干扰

EdgeVision 通过可配置化 Pipeline 引擎、多模态算法融合和边云协同架构，为工业视觉检测提供了一套完整的解决方案。系统具有**低成本、易配置、可扩展**的特点，能够有效降低视觉检测的部署门槛，提升产线质检效率。
